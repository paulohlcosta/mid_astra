Fazer a leitura dos sinais do computador de bordo MID do Astra - 1999 a 2011.

Provavelmente você sabe de algum astra que tem o computador de bordo defeituoso. O equipamento original usa lâmpadas halógenas que geram muito calor e uma tela LCD super delicada.
Com um Arduino e algum circuito eletrônico, estou trabalhando numa alternativa para esse equipamento.
